# Nebulosa-Backend

### Instalar projeto

```bash
$ yarn install
```

### Rodando o projeto

```
## Ambiente de desenvolvimento

# create migrations
yarn knex:migrate

# development
yarn dev

## Ambiente de produção

# build
yarn build

# create migrations
yarn knex:migrate

# production mode
yarn start


### Migrations rollback
yarn knex:migrate:rollback
```
